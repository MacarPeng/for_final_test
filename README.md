# 🎬 AI 智能课堂笔记生成器 (Auto Class Notes Generator)

这是一个基于 Python 和 AI 技术的自动化工具，旨在帮助学生从录制的课程视频中，自动提取包含“常考”、“重点”等关键词的片段，并生成带有视频截图和核心知识点的 PDF 复习讲义。

## ✨ 功能特点

1. **自动语音识别**：使用 `faster-whisper` 模型对课程视频进行本地高精度语音转录。
2. **自动重点抓取（免手动关键词）**：默认采用“必须考试信号词 + AI 文本分级判断”，无需手动维护课程关键词。
3. **重要程度分级过滤**：按 `exam > focus > other` 进行收录，仅保留考点和老师强调重点，自动丢弃无关内容。
4. **低价值内容抑制**：在调用大模型前先过滤学生展示、寒暄、过渡语等低价值窗口，减少误收录。
5. **PPT 智能关联与精简**：基于证据驱动策略匹配相关页面，并对页码做合法化、去重和数量限制（最多 2 页）。
6. **精美 PDF 报告**：输出结构化重点、考点摘要与复习建议，并展示每个片段的优先级和分数。
7. **智能缓存机制**：支持对已提取的音频进行缓存，同一视频多次调试无需重复提取音频。

## 🛠️ 环境依赖与安装

本项目依赖少量的系统工具和若干 Python 第三方库。

### 1. 安装 Python
请确保你的电脑上安装了 Python 3.8 或更高版本。

### 2. 安装库依赖
在项目根目录下，打开终端（Terminal 或 PowerShell），运行以下命令安装必备的库：

```bash
pip install moviepy python-pptx pillow faster-whisper jieba opencv-python fpdf2 tqdm openai pypdf
```

`main.py` 中使用到的第三方库如下（按导入模块名列出）：

- `moviepy`
- `pptx`（对应 pip 包：`python-pptx`）
- `PIL`（对应 pip 包：`pillow`）
- `faster_whisper`（对应 pip 包：`faster-whisper`）
- `jieba`（含 `jieba.analyse`）
- `cv2`（对应 pip 包：`opencv-python`）
- `fpdf`（对应 pip 包：`fpdf2`）
- `tqdm`
- `openai`
- `pypdf`

> **注意**：
> - 代码中默认使用了 GPU 加速 (`device="cuda"`) 进行 AI 识别，建议在有 Nvidia 显卡的电脑上配置相应的 CUDA 环境。如果你没有独立显卡，可以打开 `main.py` 将 `device="cuda"` 更改为 `device="cpu"`，并把 `compute_type="int8"` 改为 `compute_type="int8"` 或 `compute_type="float32"` 以便在 CPU 上运行。
> - Windows 用户：程序生成 PDF 需要使用中文字体，默认配置路径为 `C:\Windows\Fonts\msyh.ttc`（微软雅黑），无需额外配置即可工作。

## 🚀 基础用法与步骤

### 第 1 步：准备视频文件
直接运行一次 `main.py` 或者手动在其同级目录下创建一个名为 `videos` 的文件夹，将你需要处理的课堂视频文件（支持 `.mp4`, `.mkv`, `.avi` 等）放入其中。

### 第 2 步：运行程序
在终端中执行以下命令：
```bash
.\venv\Scripts\python.exe main.py
```

### 第 3 步：选择视频
先将视频拼接：点击文件目录，输入cmd，然后输入copy /b *.ts combined_lesson.ts
程序运行后，会自动扫描 `videos` 文件夹内的视频，并在控制台打印出列表。
根据提示，输入你要处理的**视频编号**并回车。

```text
--- 发现以下视频 ---
1. 高等数学第一节.mp4
2. python编程基础_复习课.mp4
请输入要处理的视频编号: 2
```

### 第 4 步：喝杯咖啡，等待生成
程序会自动执行流水线：
- 抽取音频 (若发现已有缓存则跳过) $\rightarrow$ AI 语音转录 (带进度条) $\rightarrow$ 候选片段召回（关键词+时间采样） $\rightarrow$ AI 分级过滤与考点提取 $\rightarrow$ 关联 PPT 页面 $\rightarrow$ 渲染 PDF。

### 第 5 步：查看复习讲义
处理完成后，你可以在新生成的 `results` 文件夹中找到类似 `python编程基础_复习课_复习讲义.pdf` 的文件，即为你专属的期末复习宝典！对应的图片也会存放在 `screenshots/视频名/` 目录下。

## ⚙️ 进阶配置 (main.py)

如果想让工具更契合你的实际需求，可以编辑 `main.py` 顶部的配置项：

- `MODEL_SIZE`: Whisper 模型大小。默认为 `"small"`，如果你的显存足够大，可以改成 `"medium"` 或 `"large-v3"` 以获取更高识别准确率。
- `USE_CPU`: 是否强制使用 CPU。无独显或 CUDA 环境不可用时可设为 `True`。
- `REQUIRED_KEYWORDS`（在 `keywords_config.py` 中）：必须考试信号词列表，用于弱触发候选片段。
- `interval_sec / min_gap_sec`（在 `collect_candidate_times` 中）：控制时间采样密度与候选片段最小间隔。
- `STOP_WORDS`: 可继续维护停用词列表，进一步降低口语噪声。
