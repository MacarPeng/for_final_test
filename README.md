# 🎬 AI 智能课堂笔记生成器 (Auto Class Notes Generator)

这是一个基于 Python 和 AI 技术的自动化工具，旨在帮助学生从录制的课程视频中，自动提取包含“常考”、“重点”等关键词的片段，并生成带有视频截图和核心知识点的 PDF 复习讲义。

## ✨ 功能特点

1. **自动语音识别**：使用 `faster-whisper` 模型对课程视频进行本地高精度语音转录。
2. **重点关键词捕获**：自动扫描语音记录，定位如“考试”、“必考”、“期末”等重点关键词的时间点。
3. **上下文智能分析**：提取命中关键词前后的语音上下文，并使用 `TextRank` 算法智能提取核心知识结构（过滤无用口语）。
4. **视频自动截图**：针对每个重点时间点，利用 OpenCV 自动在视频中截取对应的 PPT 画面。
5. **精美 PDF 报告**：将上述信息（截图、核心词、原始上下文）排版并导出为精美的 PDF 格式期末讲义。
6. **智能缓存机制**：支持对已提取的音频进行缓存，同一视频多次调试无需重复提取音频。

## 🛠️ 环境依赖与安装

本项目依赖少量的系统工具和若干 Python 第三方库。

### 1. 安装 Python
请确保你的电脑上安装了 Python 3.8 或更高版本。

### 2. 安装库依赖
在项目根目录下，打开终端（Terminal 或 PowerShell），运行以下命令安装必备的库：

```bash
pip install moviepy faster-whisper jieba opencv-python fpdf2 tqdm
```

> **注意**：
> - 代码中默认使用了 GPU 加速 (`device="cuda"`) 进行 AI 识别，建议在有 Nvidia 显卡的电脑上配置相应的 CUDA 环境。如果你没有独立显卡，可以打开 `main.py` 将 `device="cuda"` 更改为 `device="cpu"`，并把 `compute_type="int8"` 改为 `compute_type="int8"` 或 `compute_type="float32"` 以便在 CPU 上运行。
> - Windows 用户：程序生成 PDF 需要使用中文字体，默认配置路径为 `C:\Windows\Fonts\msyh.ttc`（微软雅黑），无需额外配置即可工作。

## 🚀 基础用法与步骤

### 第 1 步：准备视频文件
直接运行一次 `main.py` 或者手动在其同级目录下创建一个名为 `videos` 的文件夹，将你需要处理的课堂视频文件（支持 `.mp4`, `.mkv`, `.avi` 等）放入其中。

### 第 2 步：运行程序
在终端中执行以下命令：
```bash
.\venv\Scripts\python.exe main.py
```

### 第 3 步：选择视频
先将视频拼接：点击文件目录，输入cmd，然后输入copy /b *.ts combined_lesson.ts
程序运行后，会自动扫描 `videos` 文件夹内的视频，并在控制台打印出列表。
根据提示，输入你要处理的**视频编号**并回车。

```text
--- 发现以下视频 ---
1. 高等数学第一节.mp4
2. python编程基础_复习课.mp4
请输入要处理的视频编号: 2
```

### 第 4 步：喝杯咖啡，等待生成
程序会自动执行流水线：
- 抽取音频 (若发现已有缓存则跳过) $\rightarrow$ AI 语音转录 (带进度条) $\rightarrow$ 检索关键词 $\rightarrow$ 提取截图 $\rightarrow$ 渲染 PDF。

### 第 5 步：查看复习讲义
处理完成后，你可以在新生成的 `results` 文件夹中找到类似 `python编程基础_复习课_复习讲义.pdf` 的文件，即为你专属的期末复习宝典！对应的图片也会存放在 `screenshots/视频名/` 目录下。

## ⚙️ 进阶配置 (main.py)

如果想让工具更契合你的实际需求，可以编辑 `main.py` 顶部的配置项：

- `KEYWORDS`: 你可以自由修改想要捕捉的提示词。例如英语课可以改成 `["important", "exam", "test"]` 等。
- `MODEL_SIZE`: Whisper 模型大小。默认为 `"small"`，如果你的显存足够大，可以改成 `"medium"` 或 `"large-v3"` 以获取极高的识别准确率。
- `STOP_WORDS`: 如果发现生成的讲义中混进了一些老师的口头禅（如“同学们”、“然后”），可以将其加入到这个停用词列表中自动过滤。
